# To-Do App

## Description
A simple and user-friendly To-Do application that allows users to add, manage, and track their daily tasks. This app helps users stay organized and improve productivity.

## Features
- Add new tasks
- Mark tasks as completed, active and overdue
- Delete tasks
- Display number of remaining tasks
- Simple and clean user interface
- drag and drop tasks to reorder
- adding subtasks for each tasks

## Technologies Used
- HTML
- CSS
- JavaScript
